 
# UBL Digital Token Management System

A robust, corporate-standard Digital Branch Operations Kiosk backend and frontend integration mimicking live queue analytics at United Bank Limited (UBL). Built using Python Flask microservices and responsive component modules, this application transitions client-side interaction dependencies into automated server-side states.

---

## 🚀 Core Functionalities

- **Automated Sequential Matrix:** Dynamically computes distinct counter formats (`CSH-001`, `ACC-001`) without storage overlap.
- **Asynchronous Live Pipeline:** Leverages JS Promises (`fetch`) to synchronize operational dashboards in real-time without reloading interfaces.
- **Persistent Branch Operations MIS:** Tracks running queues across specific distinct counters (Cash, Accounts, Loans, Ameen Services) concurrently.
- **Validations Routing:** Built-in network exception structural mapping returning clean operational JSON logs to terminals.

---

## 🛠️ Architecture & Core Dependencies

- **Frontend Core:** HTML5, CSS3 Architecture (Dynamic Variable Branding Mapping), Asynchronous Vanilla JS.
- **Backend Microservices:** Python 3.8+, Flask Framework engine.
- **Access Policies Security:** Flask-CORS Middleware injection layer handling explicit loop blocks safely.

## ⚡ Setup & Local Execution Workflow
Execute these chronological tasks to initiate the server terminal instance locally on your machine environment:
### 1. Verification of Workspace
Ensure your environment paths have Python 3.8+ properly mapped into system environment parameters variables.
### 2. Dependency Resolution Pipeline
Open your respective Terminal or Command Prompt window inside the workspace path and trigger the package allocation manager:
```bash
pip install -r requirements.txt

```
### 3. Initialize Server Runstream
Launch the WSGI local host routing runtime process thread handler using the standard application initialization scripts:
```bash
python app.py

```
### 4. Interface Port Mapping
Open any web browser layout utility (Chrome, Edge, Firefox) and connect securely to the local application port address network channel:
```text
[http://127.0.0.1:5002](http://127.0.0.1:5002)

```
